# Ghostty 快速参考卡

## 🎯 最常用（必记）

| 操作 | 快捷键 |
|------|--------|
| 右侧分屏 | `Cmd + Alt + D` |
| 放大当前屏 | `Cmd + Alt + Enter` |
| 清屏 | `Cmd + K` |
| Quick Terminal | `Ctrl + ~` |

## 📊 实用命令

| 命令 | 用途 |
|------|------|
| `fastfetch` | 系统信息 |
| `btop` | 资源监控 |
| `claude` | Claude Code |

## 🔧 配置文件

```bash
# 编辑配置
open ~/.config/ghostty/config

# 重载配置
Cmd + Shift + ,
```

## 💡 推荐工作流

### Claude 开发
```
1. Cmd + Alt + D     # 右分屏
2. 左: claude        # 开发
3. 右: btop          # 监控
```
