# Ghostty 工作流示例

## 工作流 1：Claude Code 开发（推荐）

### 布局
```
┌─────────────────┬──────────┐
│                 │          │
│   Claude Code   │   btop   │
│                 │          │
└─────────────────┴──────────┘
```

### 步骤
1. 打开 Ghostty
2. `Cmd + Alt + D` - 右侧分屏
3. 左边输入：`claude`
4. 右边输入：`btop`
5. 开始开发，右边监控资源

### 使用技巧
- Claude 输出长时按 `Cmd + Alt + Enter` 放大查看
- 按 `Cmd + K` 快速清屏
- 鼠标点击切换焦点

## 工作流 2：全景监控

### 布局
```
┌──────────┬──────────┐
│          │          │
│fastfetch │   btop   │
│          │          │
├──────────┴──────────┤
│                     │
│    tail -f log      │
│                     │
└─────────────────────┘
```

### 步骤
1. 左边：`fastfetch`
2. `Cmd + Alt + D` - 右边：`btop`
3. `Cmd + Alt + Shift + D` - 下方：`tail -f app.log`

## 工作流 3：多项目开发

### 布局
```
┌─────────────┬─────────────┐
│  Project A  │  Project B  │
│             │             │
└─────────────┴─────────────┘
```

### 步骤
1. 左边：`cd ~/project-a && npm run dev`
2. `Cmd + Alt + D`
3. 右边：`cd ~/project-b && npm run dev`

## 快速切换技巧

### 保存布局
- 布局自动保存
- 下次打开自动恢复

### 临时放大
- `Cmd + Alt + Enter` 放大当前屏
- 再按一次恢复

### 快速清理
- `Cmd + W` 关闭不需要的分屏
- `Cmd + K` 清屏但保留历史
